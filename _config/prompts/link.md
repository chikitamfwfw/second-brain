# Link Processing Instructions

URLからコンテンツ（記事またはYouTube動画）を受け取ります。

## 処理手順
1. **タイトル**: コンテンツの正確なタイトルを記録する
2. **要約**: 内容を200-400文字で要約する（日本語）
3. **キーポイント**: 最重要アイデアを5-7点抽出する
4. **引用**: 特に重要な文章があれば原文引用する
5. **個人的洞察**: このコンテンツから学べることを1-2文で

## YouTube動画の場合
- 書き起こしが英語の場合は日本語で要約する
- タイムスタンプの参考情報があれば記録する

## 出力形式
提供されたテンプレートを使用してください。
