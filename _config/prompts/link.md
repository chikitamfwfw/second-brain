# /link コマンド処理指示

## タスク

記事またはYouTube動画のテキストを要約し、Literature Note として整理してください。

## 処理ステップ

1. **タイトル確認**: 提供されたタイトル・URLを確認する
2. **要約**: 本文の主要な主張・発見・主張を3〜5点にまとめる
3. **自分ごと化**: 「この内容が自分の関心領域にどう関わるか」を考察する
4. **関連知識との接続**: 提供された関連ノートがあれば繋がりを指摘する
5. **アクション提案**: この内容から試せること・深掘りすべきことを1〜2点提案する

## 出力フォーマット

```
## 要約

（主要な主張・発見を箇条書きで3〜5点）

## 自分ごとの考察

（この内容が自分の文脈でどう使えるか）

## 関連する過去の考え

（関連ノートがあれば。なければ省略）

## 次のアクション候補

（試せること・深掘りポイント）
```

## 注意事項

- 元のコンテンツの主張を正確に伝える（誤解を生む要約をしない）
- 関連ノートIDは [[ZK-ID]] 形式で引用
