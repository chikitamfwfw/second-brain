# 社内向けAI勉強会企画メモ（VSCode活用・プロデューサー向け）

## 要約

音楽・映像系プロデューサー（一部事業プロデューサー含む）を対象に、VS Code + AIを使った業務効率化の社内勉強会を企画中。単なるAI使用ではなく、フォルダ整理・企画書下書き・LP作成などの実務ユースケースをハンズオン形式で体験させることが目的。

## キーポイント

- 対象者: 音楽・映像系プロデューサーがメイン、事業・IPプロデューサーも含む
- ITリテラシー: Notionは使えるがGitHubは未経験、メイン作業はExcel・メール
- やらせたいこと: フォルダ整理、企画書・台本の下書き生成、LP作成
- GitHubは今回の勉強会では無理に触らせなくていい（ローカル完結で十分）
- 最初に「費用対効果」を見せることが重要。プロデューサーは「自分の仕事に使える」と思わないと脱落する

## 企画・制作への示唆

- VS Codeを「エンジニアのツール」ではなく「AIと話せる作業環境」として再定義して見せる必要がある
- 構成案: ①デモ（なぜVS Code + AI？30分→3分の体験談）→ ②ハンズオン（フォルダ整理・企画書叩き台・LP生成）
- Notionユーザーならインターフェースへの抵抗は低いため、導入コストは想定より低い可能性

## アクションアイテム

- 参加人数・開催時間・PC環境（各自持参 or 会社支給Windows等）を確定する
- ハンズオン用の具体的なデモシナリオを設計する
- VS Code + Claude Code（またはCursor）の環境構築手順を参加者レベルに合わせて整理する
